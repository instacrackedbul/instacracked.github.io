Wheelchair Simulator

ABOUT THIS GAME

We created this game to help real-world wheelchair users here, in Ukraine, where accessible infrastructure is still something to be desired. Profits from the game are donated to charity! Have I seen this game before? This is a non-VR version of the «Wheelchair Simulator VR». Put yourself in other people’s shoes (or, in this particular case, a wheelchair) even without buying the expensive VR helmet! So what's new? We completely remade the controls and made the game even more challenging. Also we added new mechanics, unexpected obstacles, and a whole new bonus level, but the heart of the game - its true story, remained intact.
